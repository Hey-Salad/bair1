The Problem

9 out of 10 people breathe polluted air

  • Thousands of London schools are near roads exceeding legal pollution limits
  • Existing monitors cost $200+ and display raw numbers nobody understands
  • Most people have zero visibility into the air inside their homes
  • No affordable, friendly consumer solution exists

UK deaths/year from air pollution
$200+

0

2

The Solution

A bear that sniffs the air for you

  • Plug in. Connect to WiFi or cellular. Done.
  • One glance. Bear's expression = air quality. No numbers needed.
  • Real data. PM1, PM2.5, PM10 from a Sensirion SPS30 laser sensor
  • Always connected. Cellular (2G) means it works without WiFi
  • Dashboard + App. Deep-dive when you want, at bair1.live or on mobile
3

What We Built This Weekend

Silicon to screen — full stack

Hardware / Firmware

  • Seeed XIAO MG24 Sense MCU
  • Sensirion SPS30 PM sensor (I2C)
  • SIM800L cellular module
  • SSD1306 OLED display
  • PlatformIO / C++ firmware

Cloud Infrastructure

  • AWS API Gateway (TLS 1.0 bridge)
  • Cloudflare Worker relay
  • Vercel Next.js 16 API + dashboard
  • Neon Postgres (serverless)
  • Auth0 + Stripe + Mapbox

Web Dashboard

  • Real-time AQI gauge + bear
  • PM1/PM2.5/PM10 breakdown
  • Historical charts
  • Mapbox sensor map
  • Tailwind v4 dark theme

Mobile App

  • React Native + Expo
  • Built on Bilt.me
  • 5-tab UI (Home, Analytics, Map, Chat, Profile)
  • Same live backend
  • iOS + Android ready
